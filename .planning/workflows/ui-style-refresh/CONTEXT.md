# CONTEXT.md — ui-style-refresh 迭代 1

## 风格方向
- **保持现有红色主题**（#ff6034 → #ee0a24 渐变），不做配色替换
- **优化 4 个维度**：间距、圆角、字号、阴影

## 现状分析

### CSS 架构
- `css/common.css`：344 行，定义 19 个 CSS 变量（但全部未被 var() 引用，仅作文档）
- 64 个页面全部引用 common.css，且全部有页内 `<style>` 块
- 149 处内联 `style=` 属性
- `page-index.html` 不引用 common.css，有独立样式

### 混用问题

**间距（gap/margin/padding）**：8px、10px、12px、14px、16px、20px、24px 混用
→ 收敛为：4px、8px、12px、16px、24px、32px（4px 递增）

**圆角（border-radius）**：4px、6px、8px、10px、12px、14px、16px、20px、22px 混用
→ 收敛为：4px（小组件）、8px（按钮）、12px（卡片）、16px（大卡片/弹窗）、24px（头像）

**字号（font-size）**：10px~24px 共 12 档混用
→ 收敛为：10px（辅助）、12px（次要）、14px（正文）、16px（小标题）、18px（标题）、20px（大标题）、24px（页面标题）

**阴影（box-shadow）**：0 1px 3px、0 2px 8px、0 4px 12px、0 8px 24px 混用，部分无阴影
→ 收敛为：sm(0 1px 3px)、md(0 2px 8px)、lg(0 4px 12px) 三级

## 优化策略

### 核心方法
1. **common.css 变量化**：将收敛后的规范写入 CSS 变量，所有组件改为 var() 引用
2. **全局替换**：按规范批量替换各页面的硬编码值
3. **逐页验证**：按任务队列分批执行，每批完成后验证

### 不改动的部分
- 主色（#ff6034、#ee0a24）保持不变
- 页面功能和布局结构不变
- 内联 style 中动态值（display:none、gradient 覆盖等）保持不变
